Paper MCP 아트보드 2개:
1. **Desktop** (1440×900) — 첫 화면 (지도 + 안내)
2. **Mobile** (390×844) — 첫 화면 (지도 + 하단 탭바)

## 성공 기준

- [ ] 회색 한국 지도가 화면 전체를 채움
- [ ] 드래그앤드롭 안내가 지도 위에 자연스럽게 오버레이됨
- [ ] PC: 얇은 상단바에 로고 + 설정 아이콘
- [ ] Mobile: 하단 탭바 (지도 / 설정)
- [ ] 초록 팔레트 색상 (`#388E3C` 계열)이 브랜드 컬러로 사용됨
- [ ] Evaluator 채점 8/10 이상

## 디자인 방향 (Generator 제안)

**무드:** 지도 탐험 도구. 깔끔하고 기능적, 그러나 차갑지 않게.

**레퍼런스 느낌:**
- Mapbox Studio 같은 다크+그린 톤 UI (but 라이트 모드)
- 지도가 주인공, UI는 보조

**구체적 결정:**
| 항목 | 선택 |
|---|---|
| 배경 | 흰색 — 지도가 돋보이게 |
| 브랜드 컬러 | `#2D6A4F` (딥 그린, 팔레트와 통일) |
| 상단바 높이 | 48px, 아주 얇게 |
| 폰트 | Inter — 깔끔한 시스템 폰트 |
| 로고 | "SnapRoute" 텍스트 + 핀 아이콘 (심플) |
| 안내 오버레이 | 지도 중앙, 반투명 흰 카드 |
| 업로드 버튼 | 초록 outline 버튼 (강조 없이 미니멀) |

**피해야 할 것:**
- 흰 카드 + 보라 그라디언트 (AI 기본값)
- 과도한 그림자, 블러 효과
- 지도를 가리는 큰 UI 요소

## Evaluator 채점 기준 (디자인 전용)

| 기준 | 가중치 | 체크 포인트 |
|---|---|---|
| **Originality** | 높음 | AI 기본값 없음? 고유한 결정의 증거 있음? |
| **Design Quality** | 높음 | 색상/타이포/레이아웃이 하나의 무드로 통일? |
| **Craft** | 낮음 | 간격, 정렬, 명암비 기본 충족? |
| **Functionality** | 낮음 | 사용자가 업로드 방법을 즉시 이해할 수 있음? |
