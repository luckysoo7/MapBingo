# SnapRoute 추후 구현 백로그

> MVP 이후에 할 것들. 잊지 않으려고 여기 모아둠.
> 우선순위는 위에 있을수록 높음.

---

## Phase 2 (MVP 직후)

- **행정구역 클릭 → 사이드 패널**: 사진 썸네일 그리드 + 날짜 목록 + 원본 미리보기 (현재는 통계 카드만)
- **읍/면/동 지원**: 줌 인 시 읍/면/동 단계까지 표시 (현재는 시/군/구까지만)
- **스냅샷 공유 기능**: 지도 전체 캡처 → PNG 저장 → 인스타 공유용
- **HEIC(iPhone) 지원**: 현재는 스킵 + 안내만, 추후 서버사이드 변환 또는 WebAssembly HEIC 디코더

## Phase 3

- **위성 지도 뷰**: 기본 미니멀 맵 외에 위성 사진 맵으로 전환 옵션
- **지역 클로즈업 스냅샷**: 특정 도시/지역 확대 + 그 지역 사진 조합 이미지
- **통계 인포그래픽 카드**: "나는 N개 시/군/구, 총 N일 방문" 공유용 카드
- **사진 썸네일 핀 모드**: 지도에 사진 미니 이미지가 직접 핀처럼 박히는 뷰 (토글)
- **안개 걷힘 효과 뷰**: 안 간 곳 어둡게, 간 곳만 선명하게 (Fog of World 스타일 토글)

## Phase 4 (고급 기능)

- **색상 테마 변경**: 초록 이외 팔레트 선택 옵션
- **카메라 타임존 설정**: 사용자가 "내 카메라는 UTC+9" 수동 지정
- **기간 필터**: "2024년만 보기", "최근 1년" 같은 날짜 범위 필터
- **해외 시/도 단계 지원**: 현재 해외는 국가 단위만, 추후 일본 도도부현 등 추가
- **안드로이드 앱 버전**: PWA 또는 React Native
